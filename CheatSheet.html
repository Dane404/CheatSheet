CSS

Name the components of a css rule properly
A css rule is build of three components. A selector, a property and a value ( selector{property: value;})

Use combinators properly: direct child, descendant, adjacent sibling, sibling
direct child:
It selects all elements that are the children of a specified element.(Combinator = >)
descendant selector:
It matches all elements that are descendants of a specified element.(Combinator = space)
adjacent sibling:
It selects all elements that are the adjacent siblings of a specified element. Sibling elements must have the same parent element, and "adjacent" means "immediately following".(Combinator = +)
General sibling:
It selects all elements that are siblings of a specified element.(Combinator = ~)
Syntax:
specified element Combinator element

Specify colors by color names and in hexadecimal notation
A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color. All values must be between 00 and FF.
A color name is specified with the name of the color(red,blue,powderblue,tomato etc.).

Box Model

Block elements have which kind of characteristics?
Block level elements always start on a new line, can have customized margin and padding, can contain other block or inline element, takes up full width of its container and the height is adjusted automatically to fit all the content.
Inline elements have which kind of characteristics?
Inline elements doesn´t require a new line to start, top and bottom margin can not be applied to inline elements and they can only contain data or other inline level elements.
What is the margin of an Html element? Where can one find it?
The margin property defines the space around an HTML element.
What is the padding of an Html element? How does it differ from margin?
The padding property defines the space inside of an element other than margin which defines it outside.
